# 에이전트 자율 워크플로우 지침 (Workspace Level Rules)

본 규칙은 Antigravity IDE 에이전트가 본 프로젝트 내에서 작업을 수행할 때 반드시 준수해야 하는 자율 행동 규칙입니다.

---

## 1. 자율적인 자가 검증 (Self-Validation Loop)
- **무조건적 빌드 검증**: 파일 수정, 신규 기능 추가, 디버깅 등 코드를 수정한 직후에는 **사용자의 추가 지시를 기다리지 않고** 즉시 빌드/테스트 명령어를 자동 실행한다.
- **본 프로젝트의 검증 명령어**:
  ```powershell
  python report.py
  ```
- **자가 디버깅 (Self-Correction)**: 빌드 또는 실행 중 에러가 발생한 경우, 일일이 사용자에게 질문하지 않고 에러 로그를 파악하여 코드를 스스로 수정한 후 다시 검증 명령어를 재실행하는 자율 디버깅 루프를 돌린다.

## 2. 코드 품질 및 의존성 자율 관리
- **코드 스타일 정돈**: 파이썬 파일 수정을 완료한 후에는 불필요한 import 구문을 지우고, 코드 가독성을 위한 기본적인 포맷팅 정돈을 완료한 후 저장한다.
- **의존성 자동 해결**: 패키지 미설치로 인한 실행 에러(`ModuleNotFoundError`)가 감지되면, 사용자의 개입을 기다리지 않고 자율적으로 `pip install` 또는 `pip install -r requirements.txt`를 실행하여 환경을 복구한다.

## 3. 디자인 및 기능 설계: 디폴트 하네스(Default Harness) 우선 적용
- **질문 최소화 (Zero-Question Mindset)**: 기능 요구나 디자인 개선 요청이 들어왔을 때, 세부적인 컬러 톤, 컴포넌트 구조, 라이브러리 선택 등을 일일이 질문하여 사용자의 확인을 받으려 하지 않는다.
- **검증된 프리미엄 디폴트 적용**:
  - 업계에서 널리 쓰이고 만족도가 높은 **모던 HSL 다크 테마(Slate/Zinc 계열)**, **글래스모피즘(Glassmorphism)**, **부드러운 마이크로 애니메이션**을 기본 테마로 즉시 적용한다.
  - 가독성을 위한 표준 모던 타이포그래피(Inter, Outfit 등 Google Fonts) 및 유연한 모바일 반응형 Flexbox/Grid 레이아웃을 디폴트 하네스로 로드하여 설계에 즉각 반영한다.

## 4. 맥락 제어 및 세션 자동 백업/복구 (Context & Session Safety)
- **맥락 자동 압축 (Context Compression)**:
  - 대화 이력이 너무 길어져 토큰 소비가 크거나 반응이 느려진다고 감지될 경우, 에이전트는 사용자의 지시 없이도 지금까지의 작업 진행 상황, 수정한 파일 목록, 남은 할 일(TODO)을 핵심 요약한 '맥락 압축본'을 스스로 생성하여 프롬프트에 주입하고 불필요한 대화 데이터를 메모리에서 정리한다.
- **주기적 세션 백업 (`context-save`)**:
  - **15분 주기**로 작업이 지속되거나, 대규모 아키텍처/코드 수정 작업에 들어가기 직전, 혹은 성공적으로 빌드/릴리즈가 완료될 때마다 **`context-save`** 스킬 또는 관련 매니저를 호출하여 세션 상태(현재 진행 상황, 계획 및 변수 등)를 로컬 백업 공간에 자동으로 저장한다.
- **세션 자동 복구 (`context-restore`)**:
  - 예기치 못한 크래시, 터미널 강제 종료 또는 연결이 끊겼다가 다시 세션이 시작될 경우, 에이전트는 다른 작업을 수행하기 전에 우선적으로 **`context-restore`** 스킬을 구동하여 직전 백업 시점의 맥락을 완벽히 복원한다.
- **룰 파일 자동 교차 생성 및 동기화**:
  - 프로젝트 루트에 `GEMINI.md`와 `CLAUDE.md` 중 하나라도 누락되어 있다면 자동으로 누락된 파일을 생성하고, 두 파일 중 하나가 수정될 경우 즉시 다른 파일에 동일한 규칙을 교차 반영하여 항상 100% 동기화 상태를 강제한다.

## 5. 자동 Git 커밋 & 릴리즈 (Auto Git Commit & Push)
- **성공 시 자동 이행**: 자가 검증(`python report.py`)이 에러 없이 깔끔하게 성공적으로 완료되면 즉시 Git 커밋 단계를 자동 실행한다.
- **커밋 메시지 자동화**:
  - 변경 사항의 본질을 요약한 한 줄의 간결하고 명확한 한국어 또는 영어 커밋 메시지를 스스로 생성한다.
- **원격 푸시 자율화**: 커밋이 생성되면 즉시 원격 저장소로 `git push`를 자율적으로 실행하여 릴리즈 및 배포 파이프라인(GitHub Actions)을 트리거한다.

## 6. 윈도우 환경 및 학습 규칙
- **PowerShell 한글 깨짐 방지**: PowerShell에서 한글 출력이 포함된 명령어를 구동하거나 파일 로그를 파악할 때, 인코딩 에러를 방지하기 위해 UTF-8 설정을 항상 고려하여 명령어를 조합하거나 실행한다.
- **영구 학습 연계 (`LEARNINGS.md`)**:
  - 개발 중 겪은 고유한 버그의 해결책이나, 사용자가 지정해준 코딩 취향/스타일은 작업 완료 후 즉시 **`.agents/LEARNINGS.md`** 파일에 누적 기록하여 다음 작업 시 이를 참고하도록 한다.
